# Звіт до лабораторної роботи 8

## Тема

Автоматизоване тестування web-сторінки каталогу товарів.

## Мета

Створити набір автоматизованих UI-тестів, які перевіряють відкриття сторінки, наявність товарів, коректність даних, сортування та роботу елементів фільтрації.

## Використані засоби

- Node.js
- Playwright Test
- HTML fixture: `fixtures/jackets.html`

## Реалізовані тестові сценарії

1. Перевірка відкриття сторінки, заголовка `Men Jackets` і кількості товарів.
2. Перевірка назв товарів та їхніх цін.
3. Перевірка сортування товарів за ціною від меншої до більшої.
4. Перевірка сортування товарів за назвою в алфавітному порядку.
5. Перевірка наявності чекбоксів фільтрів та можливості вибрати фільтр.

## Запуск тестів

```bash
npm test
```

## Результат виконання

Після запуску `npm test` усі тести виконались успішно:

```text
5 passed
```

## Висновок

У межах лабораторної роботи було створено автоматизовані UI-тести для сторінки каталогу. Тести перевіряють основні елементи інтерфейсу та поведінку сортування, що дозволяє швидко виявляти регресії у функціональності сторінки.
